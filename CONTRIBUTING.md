# 为币安慈善项目做贡献

感谢您对本项目的兴趣！我们欢迎任何形式的贡献。

## 如何贡献

### 报告 Bug

如果您在运行中发现任何错误，请通过 [Issues](https://github.com/BinanceCharity/BCT/issues) 页面提交您的问题。请尽可能详细地描述问题和复现步骤。

### 功能建议

如果您有任何关于新功能或改进的建议，也请通过 [Issues](https://github.com/BinanceCharity/BCT/issues) 页面告诉我们。请使用 "enhancement" 标签标记功能建议。

### 提交更改 (Pull Request)

如果您想直接贡献代码：

1. **Fork 本仓库**
   - 点击 GitHub 页面右上角的 "Fork" 按钮

2. **创建新分支**
   ```bash
   git checkout -b feature/YourAmazingFeature
   # 或 bugfix/FixSomething
   # 或 docs/UpdateDocumentation
   ```

3. **进行您的修改**
   - 确保代码遵循项目规范
   - 添加必要的测试
   - 更新相关文档

4. **提交您的更改**
   - 使用清晰的提交信息
   ```bash
   git commit -m 'feat: Add some AmazingFeature'
   # 或 fix: Fix bug in something
   # 或 docs: Update API documentation
   ```

5. **推送到您的 Fork**
   ```bash
   git push origin feature/YourAmazingFeature
   ```

6. **创建 Pull Request**
   - 在 GitHub 上创建新的 Pull Request
   - 详细描述您的更改
   - 等待代码审查

## 开发指南

### 环境设置

```bash
# 克隆仓库
git clone https://github.com/BinanceCharity/BCT.git
cd BCT

# 安装依赖
npm install

# 安装 Python 依赖
pip install -r requirements.txt
```

### 代码规范

- 使用 ESLint 进行 JavaScript/TypeScript 代码检查
- 使用 Black 进行 Python 代码格式化
- 遵循项目中已有的代码风格

### 测试

- 为新功能编写单元测试
- 确保所有测试通过后再提交
```bash
npm test
```

### 文档

- 更新 README.md 如果需要
- 为新功能添加文档注释
- 保持文档与代码同步
