# 贡献者行为准则

## 我们的承诺

作为项目成员、贡献者及领导者，我们承诺将参与体验打造为无骚扰的环境，无论年龄、体型、可见或不可见的缺陷、种族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性认同与性取向。

我们承诺在与项目相关的所有交互中以促进开放、友善、多元、包容与健康社区的方式行动。

## 我们的标准

有助于创造积极环境的行为示例包括：

* 对他人表现出同理心与善意
* 尊重不同观点、看法与经验
* 给予并优雅地接受建设性反馈
* 承担责任并为错误向受影响者道歉，汲取经验教训
* 不仅关注个人利益，更关注整体社区的最佳利益

不可接受的行为示例包括：

* 使用性化的语言或图像，以及任何性关注或性企图
* 挑衅、侮辱或贬损评论，以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可发布他人私人信息，如物理或电子邮箱地址
* 其他可被视为职业环境中不适当的任何行为

## 执行责任

社区领导者有责任澄清并执行我们可接受行为的标准，并对他们认为不适当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

社区领导者有权并有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题及其他贡献，并在适当时机告知采取措施的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于在公共空间中代表社区的个人。代表社区的示例包括使用官方电子邮件地址、通过官方社交媒体账户发布信息，或在线上/线下活动中担任指定代表。

所有社区领导均有义务尊重任何事件报告者的隐私与安全。

## 执行指南

社区领导者将遵循以下社区影响指南，以确定他们认为违反本行为准则的任何行为所引发的后果：

### 1. 纠正

**社区影响**：使用不当语言或其他在社区中被视为不专业或不受欢迎的行为。

**后果**：社区领导者的私下书面警告，明确违规性质并解释行为不当的原因。可能被要求公开道歉。

### 2. 警告

**社区影响**：通过单次事件或一系列行为造成的违规。

**后果**：带有继续行为后果的警告。在指定时间内禁止与涉及人员进行互动，包括与执行行为准则的人员进行主动互动。这包括在社区空间以及外部渠道如社交媒体中的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定期限内禁止与社区进行任何类型的互动或公开沟通。在此期间禁止与涉及人员进行公开或私下互动，包括与执行行为准则的人员的主动互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、骚扰个人，或针对整个群体的攻击或轻视。

**后果**：永久禁止在社区内进行任何公开的互动。
